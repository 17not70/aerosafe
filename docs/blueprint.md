# **App Name**: AeroSafe Insights

## Core Features:

- VSR/MOR Submission: Structured forms for submitting Voluntary Safety Reports (VSR) and Mandatory Occurrence Reports (MOR) with attachment support.
- Risk Assessment Engine: Automated risk index calculation (Severity x Probability) using Cloud Functions.
- Corrective Action Tracking: Track the status of corrective and preventative actions associated with each safety report.
- Role-Based Access Control: Firebase Authentication with role-based permissions (Reporter, Safety Officer, Admin) to manage access to reports and features.
- Interactive Dashboards: Visualize safety data through interactive dashboards to monitor safety performance trends.
- Data Export: Export safety data in CSV/PDF format for regulatory reporting and further analysis.
- Smart Recommendations: Use generative AI as a tool to provide recommendations based on the information in the reports, for potential corrective actions or areas of concern.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: Orange (#FF9800) to highlight critical actions and alerts.
- Body text: 'PT Sans', a sans-serif font for clear, accessible text. 
- Headline Font: 'Space Grotesk', a sans-serif font for a scientific feel, suitable for headlines.
- Use industry-standard aviation safety icons to enhance usability.
- Implement a clean and intuitive layout using React and Tailwind UI components, focusing on data presentation and accessibility.
- Subtle animations to indicate data loading and confirmation of actions.
